# 💰 Java Bank System

A simple banking system developed in Java using Object-Oriented Programming principles.

## 🚀 Features

* Create bank accounts
* Deposit money
* Withdraw money
* Transfer between accounts
* Password authentication system
* Attempt limit for security
* Transaction history tracking
* Different account types (Standard and Deluxe)

## 🧠 Concepts Used

* Encapsulation
* Inheritance
* Polymorphism
* Business rules validation
* Basic security logic (password system)

## 📂 Project Structure

* `ContaBase` → base class for all accounts
* `ContaPadrao` → standard account with lower limits
* `ContaDeluxe` → premium account with higher limits

## ⚙️ How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the `Main` class
